# tanksgame

## Prerequirements

### Linux
Installed packages:
```
libsdl2-dev
libsdl2-net-dev
libsdl2-image-2.0-0
libsdl2-image-dev
libsdl2-ttf-dev
```
### Windows
mingw64 libraries installed in:
```
C:\tools\mingw64
```
### Mac OS
Installed frameworks:
```
/Library/Frameworks/SDL2
/Library/Frameworks/SDL2_image
/Library/Frameworks/SDL2_net
```

## Building

### Linux & MacOS
Simply enter the project folder and type
```
make
```
### Windows
CLI
```
make
```
